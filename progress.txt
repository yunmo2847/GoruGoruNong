ARCHITECTURE DECISIONS
- Hybrid data lifecycle (plan principle #5): baseline/market-stats = frozen snapshot
  (seed script output, src/data/*.json). User plans = live, file-backed (server/store).
- Backend: Express + TypeScript run via tsx; port 5174. Vite dev proxies /api -> 5174.
- Auth: bcryptjs password hash + JWT in httpOnly cookie. Secret server-side only (.env).
- Map: point/bubble choropleth colored by RiskTier over a Korea backdrop (offline-safe;
  no leaflet tile network dependency). README documents dropping in a simplified GeoJSON.
- No real KAMIS/KOSIS/MAFRA keys available -> seed script uses documented deterministic
  fallback generator producing realistic frozen snapshot (header comment documents this).

CROP TAXONOMY (부류)
  엽채류: 배추 상추 시금치 양배추 | 근채류: 무 당근 감자 고구마
  양념채소: 양파 마늘 대파 고추 생강 | 과채류: 토마토 오이 호박 딸기
  과수: 사과 배 감귤 포도 | 곡물: 콩 쌀
